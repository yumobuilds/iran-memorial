
IRAN MEMORIAL
A digital memorial built to honour and remember the lives lost in Iran. Thousands of names — real people — displayed so the world can see and never forget.
Built by YUMO BUILDS
---
🕯️ About
This is not a political project. It is a human one. Every name in this memorial was a person with a life, a family, and a future. This site exists so their names are not lost — freely hosted, open to anyone in the world.
